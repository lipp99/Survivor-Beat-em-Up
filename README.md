# Survivor-Beat-em-Up
Projeto criado para avaliação da Red District Studios.

Versão
Foi utilizado neste projeto a versão Unity 6.000.0.25 LTS.

Animações 
Foram utilizadas animações da mixamo e também da assest store.

Informações da gameplay
Movimentação do personagem - W A S D, setas ou Analógico esquerdo do controle.
Ataques - Click esquerdo do mouse, quadrado ou x das manetes.

Agradecimentos 
Agradeço a Red District Studios pela oportunidade de realiar este projeto-teste.
